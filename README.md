# Microservice Project

This project is a microservice-based application with account management and GraphQL functionalities.

## Running the Project

To run the project, use the following command:

```bash
docker-compose up -d
```

## Services

This project consists of the following services:

*   **Account Service:** Manages user accounts, including creation, authentication, and profile management.
*   **GraphQL Service:** Provides a GraphQL API endpoint for interacting with the application's data and functionalities.

## Project Structure

The project is organized into the following main directories:

*   **`account/`**: Contains the code for the Account service.
*   **`graphql/`**: Contains the code for the GraphQL service.
*   **`protos/`**: Contains the Protocol Buffer definitions for communication between services.
